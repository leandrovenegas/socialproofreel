## CAPABILITIES (Vercel Dashboard)
- **Template Control:** JSON-based UI to modify aesthetic variables (colors, fonts, structure) saving directly to Supabase.
- **Visual Preview (Remotion Ready):** Live simulation of the 9:16 video layout using React components and Tailwind CSS, structurally prepared to map directly as `inputProps` in Remotion.
- **Queue Management:** Real-time monitoring of video processing states (pending, rendering, completed).
- **Lead Management:** CRUD operations for the 6,000 leads database.
- **Multi-tenancy:** Capability to manage different business accounts using the Next.js starter layout.
